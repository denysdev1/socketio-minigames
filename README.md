## Tech Stack

TypeScript, jQuery, Bootstrap, Express, Socket.IO

## Run Locally

Clone the project

```bash
  git clone git@github.com:denysdev1/socketio-minigames.git
```

Go to the project directory

```bash
  cd socketio-minigames
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
